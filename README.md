## üöÄ Getting Started

### Prerequisites
- **Java Development Kit (JDK)** 8 or higher.
  - [Download Oracle JDK](https://www.oracle.com/java/technologies/downloads/) or [OpenJDK](https://openjdk.org/).
- A code editor or IDE (VS Code, IntelliJ IDEA, Eclipse).

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/subhasish20/100-Days-of-Java-Programming.git
    cd 100-Days-of-Java-Programming
    ```

2.  **Navigate to a specific day**:
    ```bash
    cd Day-1-Intro
    ```

3.  **Compile and Run**:
    ```bash
    javac hello_word.java
    java hello_word
    ```
    *(Note: Filenames may vary per day. Check the file list in each folder.)*

## üí° Real-World Use Cases

- **Byte/Short**: Used in embedded systems or file processing where memory conservation is critical.
- **Float/Double**: Essential for scientific calculations, graphics rendering, and financial applications (use `BigDecimal` for currency!).
- **Boolean**: The backbone of control flow, decision making, and feature flags in modern apps.
